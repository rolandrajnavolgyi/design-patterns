# design-patterns
Practical demonstration of SOLID principles and known design patterns.

The content of this repository is built upon the knowledge presented by Dmitri Nesteruk via his course on Udemy, which I highly recommend to watch:
https://www.udemy.com/course/design-patterns-csharp-dotnet/

During the examples I have used C# 14.0 and .NET 10.0, and skipped null checking for simplicity.
